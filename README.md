# מתכנן מערכת השעות 🗓️

פלטפורמה לתכנון ובניית מערכת השעות הבית-ספרית — שנה"ל תשפ"ז.

## מה יש כאן
- **לוח כיתות** — שיבוץ שיעורים בלחיצה על תא; שיעור משותף לכמה כיתות מופיע בכולן אוטומטית
- **לוח מורים** — כל השעות של כל מורה, כולל שהות/פרטני/תפקיד, עם סיכום חי מול המכסה (פרונטלי + פרטני + שהות)
- **זיהוי התנגשויות** — התראה כשמורה משובץ בשני מקומות באותה שעה
- **מכסות** — מעקב שעות מורים מול המכסה, ומעקב תקן שעות כיתתי לפי מקצוע (למשל: אנגלית 4, מדעים 5)
- **מערכת אישית** — תצוגה נקייה להדפסה לכל כיתה או מורה
- **גיבוי ושחזור** — הנתונים נשמרים אוטומטית בדפדפן + הורדת קובץ גיבוי

## חשוב לדעת
הנתונים נשמרים **בדפדפן של המחשב הזה בלבד** (localStorage).
כדי לעבוד ממחשב אחר — מורידים קובץ גיבוי (⬇) ומעלים אותו שם (⬆).
מומלץ להוריד גיבוי אחרי כל ישיבת עבודה משמעותית.
